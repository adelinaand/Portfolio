Plant Baby

Plant Baby is a concept and prototype created in the course Media Technology and Interaction Design. The concept of Plant Baby includes sensors that measure plant's well-being and then informs the user of what needs to be done though an app.

More information can be seen in the video below and the design process can be found in this workbook.

Project Thunderball

Project Thunderball is a project created in the course Information Visualization and was done in collaboration with Greenlytics and their weather forecast API. Thunderball allows the user to see temperature and wind speed at different times of the day, as predicted by different weather forecasts. Built with React, D3, and MapBox.

More information can be found on this website and in the video below.

solAR

solAR is a project created in the course Multimodal interaction and interfaces. It is an application that features the solar system in augmented reality (AR) as well as an informational audio track for a complementary use of multiple output modalities. The main aim of the project was to create a tool for teaching middle school children about the solar system. The application was bult in Unity with Vuforia engine.
